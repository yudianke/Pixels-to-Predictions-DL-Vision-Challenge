# Pixels-to-Predictions-DL-Vision-Challenge
The goal is to develop a SmolVLM-500M-Instruct model (from an official Hugging Face pretrained checkpoint) that uses the provided images and question context to predict the correct answer choice for each test example, with leaderboard performance measured by prediction accuracy.

## Model weights

LoRA/DoRA adapter (~16 MB):
https://drive.google.com/drive/folders/1_I1Z6IEYLI2xRKIFfkz6pDA7Qbh2PskK?usp=sharing

Base model is `HuggingFaceTB/SmolVLM-500M-Instruct`, downloaded automatically
by `transformers`.
